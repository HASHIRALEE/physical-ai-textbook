# Physical AI and Humanoid Robotics Textbook

This repository contains a comprehensive, AI-native textbook for a university-level course on Physical AI and Humanoid Robotics.

## Overview

This textbook covers fundamental and advanced concepts in Physical AI and Humanoid Robotics, designed for university-level students. The content is organized into 2000-word chapters that are self-contained yet interconnected, following the Spec-Kit Plus methodology.

Each chapter includes:
- Learning objectives
- Theoretical foundations
- Practical examples
- Python and ROS2 code snippets
- Interactive exercises
- Chapter summaries

## Structure

- `.specify/` - Spec-Kit Plus configuration and templates
- `specs/` - Feature specifications, plans, and tasks
- `docs/` - Docusaurus-based textbook content and website
- `history/` - Prompt History Records and Architecture Decision Records

## Technical Implementation

- **Framework**: [Docusaurus](https://docusaurus.io/)
- **Deployment**: GitHub Pages
- **Content**: Markdown files with embedded Python/ROS2 code examples
- **Features**: Interactive exercises, code examples, and planned RAG chatbot integration

## Getting Started

1. Clone the repository
2. Navigate to the `docs` directory
3. Install dependencies: `npm install`
4. Start the development server: `npm run start`

## Development

This project follows the Spec-Kit Plus methodology with:
- Feature specifications in `/specs/`
- Implementation plans and tasks
- Architecture decision records
- Prompt history records
