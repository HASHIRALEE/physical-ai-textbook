# Physical AI and Humanoid Robotics Textbook

This repository contains a comprehensive, AI-native textbook for a university-level course on Physical AI and Humanoid Robotics.

## About

This textbook covers fundamental and advanced concepts in Physical AI and Humanoid Robotics, designed for university-level students. Each chapter includes:

- Learning objectives
- Theoretical foundations
- Practical examples
- Python and ROS2 code snippets
- Interactive exercises
- Chapter summaries

## Structure

The textbook is organized into 2000-word chapters that are self-contained yet interconnected:

- **Introduction**: Overview of Physical AI and humanoid robotics
- **Chapters 1-9**: Core topics including locomotion, perception, control, etc.
- **Tutorials**: Practical implementation guides
- **Exercises**: Interactive assignments for each chapter

## Technical Implementation

- Built with [Docusaurus](https://docusaurus.io/)
- Deployed to GitHub Pages
- Includes Python/ROS2 code examples
- Features integrated RAG chatbot for enhanced learning

## Running Locally

1. Clone the repository
2. Navigate to the `docs` directory
3. Install dependencies: `npm install`
4. Start the development server: `npm run start`
